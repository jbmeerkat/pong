![Menu](https://i.imgur.com/xYqXY8S.png)

## Installation

Clone repo and run

`$ make install`

## Playing

### Controls

Right player: `up` and `down`

Left player: `w` and `s`

### Start the game

`$ make`

## Testing

Run linters and tests:

`$ make build`

Run test only:

`$ make test`

Run linters only:

`$ make lint`
